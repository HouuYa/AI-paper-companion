# AI-paper-companion

Personal AI Safety & Governance Knowledge Graph.

## Vision
Build a personal research companion that transforms AI papers into structured knowledge through an LLM-assisted workflow.

## Workflow
1. LLM generates a draft.
2. Human critically reviews and edits.
3. Structured knowledge is stored in Supabase.
4. Cross-paper relationships are built into a Knowledge Graph.

## Learning Sources

### AI Canon (a16z)
One of the primary learning resources for this project is AI Canon, a curated collection of foundational AI papers, talks, blog posts, books, and educational resources assembled by Andreessen Horowitz (a16z). AI Canon serves as a high-quality reading roadmap for understanding modern artificial intelligence from fundamentals to frontier research.

The papers studied in this repository will be cross-referenced with AI Canon whenever applicable.
